

| Pilha    | Entrada    | Ação       |

| FT'E' %  | id*id %    | F -> id    |
| idT'E' % | id*id %    | casamento  |

| *FT'E' % | *id %      | casamento  |
| FT'E' %  | id %       | F -> id    |
| idT'E' % | id %       | casamento  |
| T'E' %   | %          | T' -> \epsilon    |
| E' %     | %          | E' -> \epsilon    |
